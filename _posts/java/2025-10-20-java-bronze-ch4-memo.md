## No.6
**選択肢:** D×C  
**結果:** ×  
**ポイント:** 
for文で条件式を省略するとコンパイルエラーにはならないが、無限ループになる。
```java
for(int i = 0; ; i++) {
    System.out.println("Hello");
}
```
↓  
「Hello」が無限に表示される。
